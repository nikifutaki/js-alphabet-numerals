JavaScript Alphabet Numerals(base 26oid) Library

# Usage

```
npm i alphabet-numerals
```

```js
import {num2Ab,ab2Num} from 'alphabet-numerals'

const encode = num2Ab(50) // AX
const decode = ab2Num('C') // 3
```
